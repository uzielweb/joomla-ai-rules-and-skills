# Joomla AI Rules para GitHub Copilot

O Github Copilot Workspace suporta diretrizes globais através da pasta `.github`.

## Como usar

1. Copie o arquivo `.github/copilot-instructions.md` para a pasta `.github` na raiz do seu projeto Joomla.
2. O Copilot passará a gerar autocompletes, sugerir código e rodar o Chat guiado pelo padrão *Joomla Modern Standards* (sem legacy code, usando namespaces corretos e o Web Asset Manager).
