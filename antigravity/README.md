# Joomla AI Rules para Antigravity IDE

O Antigravity é extremamente poderoso por suportar "Skills" modulares que a IA lê de forma persistente. 

## Como usar

1. Copie o conteúdo da pasta `config` para o diretório raiz das configurações do Antigravity no seu sistema (geralmente `C:\Users\SEU_USUARIO\.gemini\config\`).
2. O arquivo `global-rules.md` (presente na raiz deste repositório) pode ser renomeado para `rules.md` e salvo em `C:\Users\SEU_USUARIO\.gemini\rules.md` para ativar o comportamento estrito de linting, segurança e economia de tokens.

### Arquivos Inclusos
- `config/skills/joomla/SKILL.md`: Skill avançada de Joomla Modern Standards que ensina scaffolding nativo, Web Asset Manager sem `joomla.asset.json` como requisito e padrões Ipsis Litteris.
