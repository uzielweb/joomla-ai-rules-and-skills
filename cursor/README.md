# Joomla AI Rules para Cursor

O Cursor usa o formato `.mdc` para aplicar regras de IA ao contexto do repositório automaticamente.

## Como usar

1. Copie a pasta `.cursor` que está dentro deste diretório para a raiz do seu projeto Joomla.
2. O arquivo `.cursor/rules/joomla.mdc` fará com que o Cursor Agent sempre obedeça à arquitetura MVC moderna, não utilize prefixos "J" antigos e cuide da injeção de dependências nativa do Joomla 5/6 automaticamente.
